# Dashboard Perfil de Cliente - Almeida Junior

Dashboard interativo para análise do perfil de clientes dos shoppings da rede Almeida Junior.

## Funcionalidades

- **Visão Geral**: Métricas consolidadas de todos os shoppings
- **Por Shopping**: Análise detalhada de cada unidade
- **Perfil Demográfico**: Distribuição por gênero e faixa etária
- **High Spenders**: Análise dos clientes top 10%
- **Comparativo**: Comparação entre shoppings selecionados

## Shoppings

| Sigla | Shopping |
|-------|----------|
| BS | Balneário Shopping |
| CS | Continente Shopping |
| GS | Garten Shopping |
| NK | Neumarkt Shopping |
| NR | Norte Shopping |
| NS | Nações Shopping |

## Tecnologias

- Python 3.11+
- Streamlit
- Plotly
- Pandas

## Executar Localmente

```bash
pip install -r requirements.txt
streamlit run dashboard_perfil_cliente.py
```

## Dados

Os dados estão na pasta `Resultados/` e são atualizados periodicamente.
